Chapitre 4: Internet Et Web

Le Web et Internet sont souvent confondus, mais ils ne sont pas exactement la même chose. Internet est un support de communication général, tandis que le Web est une application spécifique d'Internet. Le World Wide Web (WWW) est un système hypertexte public qui permet de consulter des pages accessibles sur des sites via un navigateur. Les pages web sont liées entre elles par des hyperliens, d'où l'image de la toile d'araignée. Le Web permet d'accéder à différents types de ressources, tels que des textes, des images, des sons, des vidéos et des contenus interactifs.

Une URL (Uniform Resource Locator) est utilisée pour accéder à une page web en ouvrant une nouvelle page. Elle contient l'adresse du serveur et l'adresse de la page sur le serveur. Par exemple, http://example.com/un/chemin/page.html. Les aspects techniques du Web comprennent trois mécanismes de base : les URL, le protocole HTTP et le langage HTML. Les URL permettent de localiser les ressources sur le Web en utilisant le protocole DNS. Le protocole HTTP est utilisé pour les communications entre les clients (navigateurs) et les serveurs web, et il fonctionne selon un modèle client-serveur sans état, ce qui signifie qu'il ne conserve pas d'informations sur les transactions précédentes. HTML est un langage de description de pages web qui permet de structurer le contenu des pages, y compris le texte, les images, les liens hypertextes et la programmation déclarative.

Une page web est un document multimédia composé d'éléments tels que des images, du texte, des liens et des possibilités d'interaction. Elle est décrite en utilisant un langage de description comme HTML, XHTML ou HTML5, et est stockée dans un fichier avec une extension .html ou .htm. Une page web est interprétée et affichée par un navigateur, et elle est accessible sur Internet via une URL. Un site web est un regroupement de pages web liées entre elles par des liens hypertextes, appartenant à la même entité et généralement hébergées sur un même serveur. Les sites web sont structurés sous forme d'arborescence de dossiers et de fichiers, et ils permettent l'utilisation de chemins relatifs pour définir les liens entre les pages.

Pour créer une page web en HTML, on utilise un langage de description appelé HyperText Markup Language. Un document HTML est un fichier texte structuré par des balises et interprété par un navigateur. Les balises HTML délimitent différentes parties du document, telles que l'entête (head), le titre (title) et le corps (body). Les titres sont définis avec les balises h1 à h6, et les paragraphes sont définis avec les balises p. Un éditeur de pages web ou un éditeur de texte peut être utilisé pour créer des fichiers HTML. En résumé, le Web est une application d'Internet qui permet d'accéder à des ressources via des pages web liées entre elles. Les pages web sont créées en utilisant le langage HTML et sont interprétées par les navigateurs. Les sites web regroupent plusieurs pages web autour d'une thématique commune et sont structurés de manière hiérarchique.
